# SuperCode Beer Shop Project

## Purpose of the website

- Create a Beer Shop
- Fetch Data from Beers API
- Dynamic Routing with React Router DOM

## Built with

- Mobile Design
- Grid and Flexbox Layout
- React
- React Router DOM
- Fetch-API
